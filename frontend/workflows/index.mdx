# Workflows

A frontend workflow spans design tokens, component design, component implementation, polish review, and accessibility audit. Each step has an artifact; each artifact gates the next step. The flow scales with the project's [wave](/backend/workflow/waves) — lighter at W1, full at W3+ — but the order stays constant.
